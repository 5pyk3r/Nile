# Nile - e-commerce app with electronics. 
# Nile made with Angular 11 and Spring
 
## Technology stack
|   Component   |   Technology  |
| ------------- | ------------- |
| Frontend      |   Angular 11  |
| Backend       |   Spring 5    |
| In Memory DB	|     H2        |

## Requirements to run the application

  - Java 14
  - Angular CLI: 10.1.3 or above
  - Node: 12.18.3 or above
  - npm: 6.14.7 or above
